# OrderBook

To build and run the project, follow these steps:

1. Create a build directory:
    ```sh
    mkdir build
    ```

2. Change into the build directory:
    ```sh
    cd build
    ```

3. Run CMake to configure the build system:
    ```sh
    cmake ../source
    ```

4. Build the project:
    ```sh
    cmake --build .
    ```

5. Run the executable:
    ```sh
    ./Orderbook
    ```
